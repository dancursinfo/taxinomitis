Collecting diagnostics

Unfortunately, students sometimes encounter errors when using the Machine Learning for Kids site.
I'm very sorry about this.
This page has some instructions for how you can collect information about what went wrong that will help me to fix it quickly, to prevent it happening again.
  1. Go to https://machinelearningforkids.co.uk/#!/welcome?debug
    (the important bit is the ?debug at the end - this should work with any page on the site)
  2. If it works, you'll see a new "DOWNLOAD LOG" item in the menu at the top of the page. (If you don't see that, please refresh the page until you do)
  3. Recreate the problem - retrace your steps to demonstrate the error that you've seen
  4. Click on DOWNLOAD LOG to download a file that will have a description of everything that you did, and the error that was encountered
  5. Please send me that file:
    • by email, or
    • by creating an issue in Github, or
    • by posting a message in the support forum
