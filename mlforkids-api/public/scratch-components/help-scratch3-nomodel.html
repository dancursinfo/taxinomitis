This block lets you use machine learning models in your Scratch projects. However, when you tried to use it, it didn't work because you don't seem to have a machine learning model.
There are a few reasons why this might have happened:
1. You haven't trained a machine learning model yet.
If this is the problem, use the train new machine learning model block to train a model.
2. You've started training a machine learning model, but it is still training and isn't ready yet.
You can use the is the machine learning model still training? block to check for this.
3. You tried training a machine learning model, but that didn't work.
You can use the is the machine learning model broken? block to check for this.
There are lots of reasons why this might happen. Maybe you don't have enough training examples yet? You can check this on the Learn & Test page for your project.
