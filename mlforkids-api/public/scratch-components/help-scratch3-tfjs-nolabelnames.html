The recognise label block lets you use TensorFlow.js models in your Scratch projects. TensorFlow.js models often include a file called metadata.json that has extra information about the model (such as this one).
The model you are using doesn't seem to have a metadata.json file. That means the Scratch blocks don't know the names of the labels the model has been trained to recognise.
That means the recognise (label) block will return values like label 10, referring to position of the label in the list of labels the model was trained to recognise. You will need to look up what this means in the documentation for the model you are using.
